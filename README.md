# Python - MEB Bilgisayar Bilimi (Kur-1)

Bu repo, **MEB başlık yapısına uygun** (menü/harita) ve öğretmenin kendi ders akışına göre güncellenebilir
bir **ömürlük ders kitabı + kod örnekleri** yapısıdır.

## Yapı
- `docs/` : GitHub Pages (MkDocs) ile yayınlanan ders kitabı
- `code/` : Sınıfta çalıştırmalık temiz örnek kodlar
- Şimdilik sadece **10. Listeler** bölümü doldurulmuştur; diğerleri “yakında” sayfalarıdır.

## İsimlendirme
Dosya adları çift dilli:
- `10_1_liste_kavrami__list_concept.md`
- `10_1_liste_kavrami__list_concept.py`
